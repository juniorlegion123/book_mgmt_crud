# Book Library - Backend Connected (React)

React app (plain CSS, card view) connected to Spring Boot backend.
Backend URL: http://localhost:8080/api/books

Features:
- Card view with modal Add/Edit
- Search by title
- Filter published books
- Delete with confirmation
- Axios for API calls

Install and run:
1. unzip and `cd book-library-backend-connected`
2. `npm install`
3. `npm start`
